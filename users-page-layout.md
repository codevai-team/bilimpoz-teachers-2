# Расположение и размеры контейнеров - Раздел "Пользователи"

Подробная документация структуры, размеров и расположения всех контейнеров на странице пользователей.

---

## Общая структура страницы

### DashboardLayout (Обертка)

**Расположение**: Корневой контейнер всей страницы  
**Размеры**:
- **Ширина**: 100% (занимает всю ширину экрана)
- **Высота**: `h-screen` (100vh)
- **Фон**: `#0b0b0b`

**Структура**:
```
DashboardLayout
├── Header (фиксированный сверху)
├── Sidebar (фиксированный слева, ширина: 288px на lg+)
└── Main Content (основной контент)
```

---

## Main Content Area (Основная область контента)

**Расположение**: Центральная область страницы  
**Размеры**:
- **Padding Top**: `pt-24` (96px) - отступ сверху для Header
- **Padding Left**: `lg:pl-72` (288px на экранах ≥1024px) - отступ слева для Sidebar
- **Padding Right**: `pr-4` (16px)
- **Padding внутренний**: `p-4` (16px на мобильных) / `lg:p-6` (24px на десктопе)

**Высота контента**:
- **Высота**: `calc(100vh - 6rem)` (высота экрана минус высота Header)
- **Overflow**: `overflow-y: auto` (вертикальная прокрутка при необходимости)

---

## Контейнер страницы пользователей

**Класс**: `<div className="space-y-6">`  
**Расположение**: Внутри Main Content Area  
**Размеры**:
- **Ширина**: 100% (занимает всю доступную ширину)
- **Отступы между элементами**: `space-y-6` (24px вертикальный отступ между дочерними элементами)

**Структура**:
```
<div className="space-y-6">
  ├── Заголовок страницы
  ├── Статистика пользователей (4 карточки)
  ├── Блок ошибки (условно)
  ├── Фильтры (UsersFilter)
  ├── Таблица пользователей (UsersTable)
  └── Пагинация (Pagination)
</div>
```

## 1. Заголовок страницы

**Расположение**: Верхняя часть страницы  
**Размеры**:
- **Ширина**: 100%
- **Высота**: Автоматическая
- **Layout**: Flexbox (`flex items-center justify-between`)

**Элементы**:

### Заголовок (h1)
- **Размер шрифта**: `text-2xl` (24px)
- **Вес**: `font-bold`
- **Цвет**: `text-white`
- **Отступ снизу**: `mb-2` (8px)

### Подзаголовок (p)
- **Размер шрифта**: Базовый (16px)
- **Цвет**: `text-gray-400`
- **Отступ**: Нет

**Код**:
```jsx
<div className="flex items-center justify-between">
  <div>
    <h1 className="text-2xl font-bold text-white mb-2">Пользователи</h1>
    <p className="text-gray-400">Управление пользователями системы</p>
  </div>
</div>
```

## 2. Статистика пользователей (4 карточки)

**Расположение**: Сразу после заголовка  
**Layout**: CSS Grid  
**Размеры контейнера**:
- **Ширина**: 100%
- **Grid**: `grid-cols-1` (мобильные) → `md:grid-cols-4` (4 колонки на ≥768px)
- **Gap**: `gap-6` (24px между карточками)

### Отдельная карточка статистики

**Размеры**:
- **Ширина**: Автоматическая (1/4 от доступной ширины на десктопе)
- **Padding**: `p-6` (24px со всех сторон)
- **Border Radius**: `rounded-2xl` (16px)
- **Фон**: `bg-[#151515]`
- **Hover фон**: `hover:bg-[#1a1a1a]`

**Внутренняя структура**:

#### Иконка и заголовок
- **Layout**: Flexbox (`flex items-center gap-3`)
- **Отступ снизу**: `mb-3` (12px)

**Иконка контейнер**:
- **Размеры**: `p-2` (8px padding)
- **Размер иконки**: `h-5 w-5` (20px × 20px)
- **Фон**: `bg-[#242424]`
- **Hover фон**: `group-hover:bg-[#2a2a2a]`
- **Border Radius**: `rounded-lg` (8px)

**Заголовок (h3)**:
- **Размер шрифта**: `text-sm` (14px)
- **Вес**: `font-medium`
- **Цвет**: `text-gray-400`
- **Hover цвет**: `group-hover:text-gray-300`

**Числовое значение (p)**:
- **Размер шрифта**: `text-2xl` (24px)
- **Вес**: `font-bold`
- **Цвет**: `text-white`

**Код**:
```jsx
<div className="grid grid-cols-1 md:grid-cols-4 gap-6">
  <button className="bg-[#151515] hover:bg-[#1a1a1a] rounded-2xl p-6 transition-all text-left group">
    <div className="flex items-center gap-3 mb-3">
      <div className="p-2 bg-[#242424] group-hover:bg-[#2a2a2a] rounded-lg transition-colors">
        <Icons.UserGroup className="h-5 w-5 text-white" />
      </div>
      <h3 className="text-sm font-medium text-gray-400 group-hover:text-gray-300">Всего</h3>
    </div>
    <p className="text-2xl font-bold text-white">1,234</p>
  </button>
  {/* ... еще 3 карточки */}
</div>
```

## 4. Фильтры (UsersFilter)

**Расположение**: После статистики/ошибки  
**Размеры контейнера**:
- **Ширина**: 100%
- **Padding**: `p-6` (24px)
- **Border Radius**: `rounded-2xl` (16px)
- **Фон**: `bg-[#151515]`
- **Отступы между секциями**: `space-y-6` (24px)

### 4.1. Секция поиска и кнопок даты

**Layout**: Flexbox (`flex flex-col sm:flex-row gap-4`)

#### Поле поиска
- **Ширина**: `flex-1` (занимает доступное пространство)
- **Padding**: `pl-12 pr-4 py-3` (48px слева, 16px справа, 12px сверху/снизу)
- **Border Radius**: `rounded-xl` (12px)
- **Фон**: `bg-[#242424]`
- **Иконка поиска**: Абсолютное позиционирование `left-4 top-1/2`, размер `h-4 w-4` (16px × 16px)

#### Кнопка очистки фильтров
- **Padding**: `px-4 py-3` (16px горизонтально, 12px вертикально)
- **Border Radius**: `rounded-2xl` (16px)
- **Фон**: `bg-[#242424]`
- **Border**: `border border-gray-700/50`
- **Иконка**: `h-4 w-4` (16px × 16px)

### 4.2. Кнопки фильтрации по дате

**Layout**: Flexbox (`flex flex-wrap gap-2`)

**Кнопка периода**:
- **Padding**: `px-4 py-2` (16px горизонтально, 8px вертикально)
- **Border Radius**: `rounded-xl` (12px)
- **Размер шрифта**: `text-sm` (14px)
- **Вес**: `font-medium`
- **Активное состояние**: `bg-white text-black`
- **Неактивное состояние**: `bg-[#242424] text-gray-400`

### 4.3. Выбор диапазона дат и времени

**Layout**: CSS Grid (`grid grid-cols-1 sm:grid-cols-2 gap-4`)

**Каждый блок (Дата от/до)**:
- **Layout**: `space-y-3` (12px между элементами)
- **Label**: `text-sm text-gray-400 mb-2` (14px, отступ снизу 8px)
- **Внутренний Grid**: `grid grid-cols-2 gap-2` (2 колонки для даты и времени)

**DatePicker и TimePicker**:
- **Ширина**: Автоматическая (1/2 от контейнера)
- **Gap**: `gap-2` (8px между элементами)

### 4.4. Основные фильтры

**Layout**: CSS Grid (`grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-4 gap-4`)

**Select компоненты**:
- **Ширина**: Автоматическая
  - На мобильных: 100% (1 колонка)
  - На планшетах (≥640px): 50% (2 колонки)
  - На десктопе (≥1024px): 25% (4 колонки)
- **Gap**: `gap-4` (16px между элементами)

**Кнопка сортировки**:
- **Min Width**: `min-w-[48px]` (48px)
- **Padding**: `px-3 py-3` (12px горизонтально, 12px вертикально)
- **Border Radius**: `rounded-lg` (8px)
- **Иконка**: `h-4 w-4` (16px × 16px)

**Код**:
```jsx
<div className="bg-[#151515] rounded-2xl p-6 space-y-6">
  {/* Поиск */}
  <div className="flex flex-col sm:flex-row gap-4">
    <div className="flex-1 relative">
      <input className="w-full pl-12 pr-4 py-3 bg-[#242424] rounded-xl" />
    </div>
  </div>
  
  {/* Основные фильтры */}
  <div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-4 gap-4">
    <Select /> {/* 4 элемента */}
  </div>
</div>
```

## Примечания

1. Все размеры указаны в Tailwind CSS классах, которые конвертируются в пиксели:
   - `p-4` = 16px
   - `p-6` = 24px
   - `gap-4` = 16px
   - `gap-6` = 24px
   - `text-2xl` = 24px
   - `text-lg` = 18px
   - `text-sm` = 14px
   - `text-xs` = 12px

2. Sidebar имеет фиксированную ширину 288px (`lg:pl-72`) на экранах ≥1024px
